# Gene Expression Cancer Classifier

Author: Caden Turnquest

## Overview
This project applies machine learning to gene expression datasets to classify different types of cancer.

Gene expression analysis is widely used in bioinformatics and biomedical research to understand disease mechanisms and identify diagnostic biomarkers.

## Dataset
The dataset contains gene expression features from the breast cancer dataset in sklearn.  
It is saved as `data/cancer_gene_expression.csv`.

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Methods
1. Data preprocessing
2. Machine learning classification using Random Forest
3. Accuracy evaluation
